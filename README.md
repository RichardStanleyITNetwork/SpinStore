# SpinStore - Vinyl Store Website

## Description
SpinStore is an online vinyl store that allows users to browse products, add them to a shopping cart, and complete checkout. The website dynamically displays products, manages a shopping cart, and uses localStorage to persist product and cart data.

## Features
- **Product Catalogue**: Products stored as objects in JavaScript, displayed dynamically on `products.html`. Each product has an "Add to Cart" button.
- **Shopping Cart**: Tracks items, quantities, subtotals, taxes, and total cost. Cart data is saved in localStorage.
- **Checkout**: Users can view order summary and complete checkout.
- **User Authentication**: Login and registration system for users.
- Commented code showing all major project requirements.
